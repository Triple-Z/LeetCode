## Description

从上到下按层打印二叉树，同一层的节点按从左到右的顺序打印，每一层打印到一行。

例如:

给定二叉树: `[3,9,20,null,null,15,7]`,
```
    3
   / \
  9  20
    /  \
   15   7
```

返回其层次遍历结果：
```
[
  [3],
  [9,20],
  [15,7]
]
```

提示：
```
节点总数 <= 1000
```

注意：本题与 [102 题](./102.%20Binary%20Tree%20Level%20Order%20Traversal%20二叉树的层序遍历.md) 相同。
